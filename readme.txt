== Description ==

hey , this plugin provide shortcode [ratex-ils] that allow you to display ils currency convert to GBP , USD, EUR , BTC , CNY

you can find the short code in ratex ils panel on your left || right

* plugin use Vue.js a ui framework
* the plugin fetch api data
